Explanation

You are looking at the petclinic, the sample application that was introduced by spring, but now build using the playframework.

Since you are seeing this page, the appliation has started successfully.

The vets link will take you to the crud section of the vets. This crud screen is the default crud system that comes with the playframework. Since we don't want everybody to see the vets section, it is also protected by a login page. This is handled by the secure module of the playframework and is also already available.

To enable the crud and security module, stop the play server and run "play deps". Then restart the play server and all should be well
